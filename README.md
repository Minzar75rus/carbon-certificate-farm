# 🌿 Carbon Certificate Farm

**Демо-версия калькулятора углеродных сертификатов**

## 🎯 Что это?
Простой инструмент для расчёта, сколько CO₂ запечатывает биомасса 
(конопля, бамбук, деревья) при использовании в строительстве.

## 🚀 Статус проекта
Это **MVP (минимально жизнеспособный продукт)**, созданный для:
- Изучения Python и работы с данными
- Привлечения внимания к теме углеродного земледелия
- Поиска единомышленников для развития
